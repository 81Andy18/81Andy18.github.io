  Linux

Linux, the full name of GNU/Linux, is a free-to-use and freely disseminated UNIX-like operating system. Its kernel was first released on October 5, 1991 by Linus Benedicte Torvalds. It is mainly affected by Inspired by the ideas of Minix and Unix, it is a POSIX-based multi-user, multi-tasking, multi-threaded and multi-CPU operating system. It can run major Unix tools, applications, and network protocols. It supports 32-bit and 64-bit hardware. Linux inherits Unix's network-centric design philosophy and is a multi-user network operating system with stable performance. Linux has hundreds of different distributions, such as community-based debian and archlinux, and commercial-based Red Hat Enterprise Linux, SUSE, Oracle Linux, etc.

In June 2021, according to the char-misc-next submission that Linux 5.14 just entered the merge queue, Linux 5.14 officially removed the RAW driver.
